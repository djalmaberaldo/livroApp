## Aplicação para gerenciar cadastros de livro

#Backend

Para desenvolvimento do backend, foi utilizado Spring Boot com Hibernate. As classes foram geradas através do JHipster.

Os dados são cadastrados em um banco chamado Livraria no MySQL. 

A API roda na porta 8081.

#Front-end

Foi utilizado HTML 5, CSS e JAVASCRIPT puro somente. É necessário abrir o arquivo index.html e começar a usar aplicação.

É possível cadastrar, editar e excluir livros.


# Configuração do ambiente

Java 8

MySQL (com schema chamado livraria)

Se necessário, altere usuário e senha do banco de dados no arquivo application-dev.yml
